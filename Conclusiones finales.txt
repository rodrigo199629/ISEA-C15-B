conclusiones:

-  Odoo nos permite crear y diseñar diferentes módulos, de manera que se ajusten a las necesidades del cliente, considerando diferentes factores.
-  A la hora de crear un módulo, es posible exportar funciones de otros módulos, reutilizando de esta forma diferentes funcionalidades de estos, esto nos ayuda a
   reducir el tiempo de desarrollo reutilizando módulos.
-  No se pudo culminar con todos los laboratorios asignados, considerando la sobrecarga de tareas acumuladas y la cantidad de tiempo disponible.
-  Odoo nos facilita diferentes herramientas con las cuales se puede implementar sistemas de ventas para diferentes entidades o empresas que necesaiten tener un control
   de lo que venden y compran, de esta forma poder reducir costos e incrementar los ingresos de la empresa.
-  Para el desarrollo del curso fue necesario aprender y comprender el funcionamiento de los módulos más conocidos en Odoo, siendo estos "suficientes" para gestionar una empresa
   de ventas, pero como ya se vio en puntos anteriores, es preferible crear un módulo ajustable a la empresa, ya que las diferentes empresas tienen requerimientos específicos
   y estos necesitan ser atendidos de manera adecuada, para así buscar el avance y desarrollo de esta ayudando a optimizar procesos administrativos.
